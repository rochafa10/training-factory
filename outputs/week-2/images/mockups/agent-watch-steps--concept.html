Watching the Agent Work: 4 Key Moments
Pay attention at each step to stay in control
1
📝
Agent writes a plan READ IT
Review the plan before the agent starts executing. Make sure it matches your intent.
2
✅
Agent asks to create a file CLICK ACCEPT
The agent needs your permission. Read the filename and purpose before accepting.
3
</>
Agent writes code LET IT FINISH
Don't interrupt mid-generation. Wait for the full output, then review the result.
4
▶
Agent asks to run a command CLICK ALLOW
Check what command it wants to run. Only allow commands you understand.
⚠
Key principle: You are always the decision-maker. The agent proposes, you approve. Never auto-accept without reading.
