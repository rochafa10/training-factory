GitHub Copilot: Three Modes
Three levels of help — from questions to building

Chat Mode

Ctrl + Shift + I
Ask questions about code, get explanations, and brainstorm ideas. Copilot answers — you decide what to do.
"What does this formula do?"

"How do I filter by date in Excel?"
Output: Text answers in the chat panel
Use when: You need to understand something before acting

Edit Mode

Select > Right-click
Select code or text, then ask Copilot to change it. You see a diff preview and approve each edit.
"Make this table sort by date descending"

"Add error handling to this section"
Output: Inline diff — accept or reject
Use when: You know exactly what to change

Agent Mode

Mode dropdown > Agent
Describe a goal. The agent plans, executes, and iterates — creating files, running commands, and self-correcting.
"Build a shift schedule template that calculates overtime automatically"
Output: Working files on your PC
Use when: You want to delegate a multi-step task
