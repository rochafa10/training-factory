The Fix Workflow
5 steps to correct any AI agent mistake
1
Identify
Spot what's wrong in the output
2
Tell Agent
Describe the specific error clearly
3
Watch Fix
Let the agent apply its correction
4
Check Diff
Review what actually changed
5
Still Wrong?
Be more specific and go back to step 2
Remember: Each iteration should include MORE specific language. "Make it better" wastes cycles. "Change the header color to #cc0000" gets it done.
