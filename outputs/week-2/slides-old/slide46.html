Activity C Template Structure Guide — Build Your Own
Essential Elements
ROLE Specific expertise + years + personality
Example: "Senior DC manager, 15 years, data-driven"
CONTEXT Include [PLACEHOLDERS] for real data
The more specific, the better the output
TASK Add technique: CoT or Few-Shot
"First..., Then..., Finally..." for reasoning
FORMAT Explicit constraints: bullets, tables, word limits
Add examples if format matters
Live Build Exercise
1 Pick a task you do weekly (standup, report, email, analysis)
2 Write each component now — ROLE, CONTEXT, TASK, FORMAT
3 Test in Bottle Rocket with real or realistic data
4 Refine until output is work-ready — no editing needed
5 Save to your playbook with a clear name and notes
