Template Output Patterns

Fill-in-Blank
"Generate a [REPORT TYPE] for [AUDIENCE] covering [TOPIC]"
Fixed Headers
"Use these exact sections: Summary, Findings, Actions, Timeline"
Length Limits
"Keep each section under 50 words" or "Total under 200 words"
Tone Match
"Match the tone of this example: [paste example]"
Output Chain
Use one AI output as input for the next prompt — build iteratively
Example in Action
Prompt
"Summarize this meeting in exactly 3 bullet points, each under 20 words"
Result
1. Q4 throughput exceeded target by 8% across all shifts

2. Two conveyor maintenance windows needed before peak season

3. New hire onboarding pace on track for November deadline
Structured outputs save editing time — the AI does the formatting, you verify the content
