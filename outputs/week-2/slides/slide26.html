Exercise 3 — Step 2

Install the Copilot Extension

Do This
  1. 1
    Find the Extensions icon in the left sidebar (four small squares) — or press Ctrl+Shift+X
  2. 2
    Search for: GitHub Copilot
  3. 3
    Click the first result: “GitHub Copilot” by GitHub
  4. 4
    Click the blue “Install” button
  5. 5
    Wait 10–15 seconds for it to finish
✓ You should see: A new Copilot icon (small robot) in the left sidebar
