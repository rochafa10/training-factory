What Is copilot-instructions.md?

.github/copilot-instructions.md
A file you create in your project folder
Copilot reads this file automatically every time you start a conversation
Tells the agent your project's rules, conventions, and preferences
Like a training manual for every new hire — always on the agent's desk
Works in all modes: Chat, Edit, and Agent
How It Works
1. You create the file once

2. Write your rules in plain English

3. Copilot reads it before every response

4. Agent follows your rules automatically

5. Update it anytime — changes take effect immediately
Markdown format  Plain English rules  No coding required
