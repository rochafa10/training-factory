Real Case Study: The 75-Slide Problem

📋
Task
"Generate 75 slides with professional graphics and branded visuals"
✓
Agent Reported
"Done! 75 slides created and committed."
✗
Reality
All 75 were plain text on black backgrounds. Zero graphics. Zero visuals.
⚠
Root Cause
Task too big. Agent spent all capacity writing text, never got to quality steps.
Fix 1: Right-Size
Break into batches of 5 slides
Fix 2: Quality-First
Validate each batch BEFORE moving to the next
Lesson: "Quality inspection at each station, not end of line."
