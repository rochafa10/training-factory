Key Principle

The Lesson: Specific Beats Vague

Vague Result
✗
“Make it better”
Agent guesses what you want. Random changes. Unpredictable output. You waste time undoing it.
Specific Result
✓
“Add a dropdown with three options”
Agent does exactly that. One change. Correct result. Done in seconds.

This is true for agents AND for the people on your team. Specific feedback gets specific fixes — every time.
