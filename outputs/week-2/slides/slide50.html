Troubleshooting MCP Install

⚠ "npx is not recognized"
➔ Install Node.js from nodejs.org, restart VS Code, try again
⚠ Server fails to start
➔ Check internet connection, try the command again
⚠ No MCP option in Command Palette
➔ Update VS Code to latest version (Help > Check for Updates). VS Code 1.99+ required.
