The Fix Workflow

1Identify -- What is wrong? ("The page scrolls -- it's too tall")
2Tell the agent specifically -- "Reduce the padding, make cards smaller, remove the footer, fit within 500px"
3Watch the agent fix it -- review the new version
4Check the diff -- what did the agent change?
5If still wrong, be even more specific -- include exact numbers: "Each card must be exactly 80px tall"
