Exercise 7 -- Part A

Part A: What Happens with Vague Instructions

Type This in Bottle Rocket
"Make it better."
What Will Happen?
Option A (Good Behavior)
Agent asks clarifying questions
"What would you like me to improve?"
Option B (Common Behavior)
Agent makes random assumptions and changes things without asking
The Lesson
"Make it better" is not useful feedback -- for an agent OR for a team member.
