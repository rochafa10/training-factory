Exercise 3 — Step 3

Sign In with GitHub

Do This
  1. 1
    Click the Copilot icon in the left sidebar
  2. 2
    Click “Sign in to GitHub”
  3. 3
    Browser opens — enter your GitHub username and password
  4. 4
    Click “Authorize” when asked
  5. 5
    Switch back to VS Code
✓ You should see: Your GitHub username in the bottom-left corner
No GitHub account? Go to github.com and sign up with your personal email.
