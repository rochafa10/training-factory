Getting AI to Output in Consistent Formats
T
Tables
"Present this as a table with columns for [X], [Y], [Z]"
Best for: Comparisons, data summaries, schedules, KPI tracking
C
Checklists
"Create a checklist with checkboxes for each step"
Best for: SOPs, audit prep, onboarding, quality control
R
Formatted Reports
"Write an executive summary with: headline, 3 key findings, recommendation"
Best for: Leadership updates, weekly reports, incident summaries
M
Markdown
"Format using Markdown with headers, bold, and bullet points"
Best for: Documentation, knowledge base articles, procedures
Specify FORMAT explicitly — don't leave it to the AI to guess
