Chat vs Agent -- The Core Difference

Chat / Bottle Rocket
You say
"Write me a report"
AI does
Writes text in the chat window
Output
Text you copy-paste
Steps
One response
Autonomy
None -- you do everything
Agent / Copilot Agent Mode
You say
"Build me a reporting tool"
AI does
Creates files, writes code, runs commands
Output
Working files on your computer
Steps
Multiple steps, asks for approval
Autonomy
High -- agent plans and executes
