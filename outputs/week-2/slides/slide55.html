Reading an Agent’s Plan

Before an agent acts, it tells you what it plans to do. Read it like a to-do list:
Agent’s Plan
📄
1. I will create a new file called dashboard.html
✎
2. I will modify template.html to add a totals row
▶
3. I will run the command open dashboard.html to preview it
Your Response
“Yes, go ahead”
Approve all steps
“Do step 1 but skip step 3”
Selective approval
“No, instead do X”
Redirect the agent
