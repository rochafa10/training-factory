How to Undo

Method 1
⌨
Keyboard Shortcut
Ctrl + Z
Undo the last change. Works in any file, any time.
Method 2
💬
Tell the Agent
"Undo those changes"
Or say: "Revert that file." The agent will reverse its own edits.
Method 3
🔄
Source Control Panel
Click ↺ next to file
VS Code Source Control panel: click the undo icon next to any changed file.
You can always undo. Nothing is permanent.
