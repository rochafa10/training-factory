Agent Quality Control

The Root Cause Framework

When an agent says "Done!" but the work is wrong, follow these 5 steps:

1
NOTICE
Compare what you asked for vs what you got
"I asked for slides with graphics, but I see only text"
2
INVESTIGATE
Look at the actual output, not just the summary
"Open the files. Read them. Check if images load."
3
DIAGNOSE
Ask: why did the agent miss? Was the instruction unclear? Task too big? Skipped steps?
4
FIX
Give a corrected, specific instruction
"Break into batches of 5. Validate each visually before moving on."
5
PREVENT
Change the process: smaller batches + quality checks at each step
