Orchestration Patterns

Sequential
Agent A → Agent B → Agent C
Use Case:
Dive deep: gather → analyze → report
Parallel
Agents A, B, C run simultaneously
Use Case:
Multi-DC comparison reports
Supervisor
Main agent delegates to specialists
Use Case:
Complex incident analysis
Human-in-Loop
Agent pauses for approval at key steps
Use Case:
Any task with data decisions
