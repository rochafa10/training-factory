Sequential Pattern - Dive Deep Example

1 👤
You provide requirements
Voice or typed instructions defining the analysis scope
↓
2 📂
Data Agent reads CSV file
Uses Agent Mode file access to read your data
↓
3 📚
Agent loads your DC context
Reads copilot-instructions.md for site-specific knowledge
↓
4 🔎
Analysis Agent finds patterns
Identifies root causes, calculates percentages, flags anomalies
↓
5 📄
Report Agent generates markdown
Formatted output with executive summary and action items
↓
6 ✅
You review and iterate
Refine the output with follow-up prompts as needed
Key Insight
Agent Mode already works this way - understanding the pattern helps you give better instructions.
Instead of "write me a report," say "first gather the data, then find root causes, then generate with executive summary."
