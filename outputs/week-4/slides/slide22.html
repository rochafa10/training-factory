Decision Framework - Which Approach for Which Task?

1
Simple
Simple Chat Prompt
Task type: One answer, no context needed
"What does OTD stand for?"
2
Advanced
Advanced Prompt (ROLE+CONTEXT+TASK+FORMAT)
Task type: Specific format, needs reasoning
"Analyze these 5 KPIs and rank by improvement priority"
3
Agent
Agent Mode
Task type: Multiple phases, file access needed
"Read the CSV, analyze trends, generate a report"
4
Full Stack
Agent Mode + MCP + Project Memory
Task type: Recurring workflow
"Weekly dive deep on missed shipments"
The 80/20 Rule
80% of time savings come from 20% of your prompts. Optimize those.
Use the shortest prompt that produces the quality you need.
