Exercise 1

Install Playwright MCP

  • 1 Check VS Code version: Help > About (need 1.99+)
  • 2 Open VS Code terminal (Ctrl+`)
  • 3 Run the installation command below
  • 4 Click "Start" on MCP server in config
  • 5 Verify tools are available in Copilot Chat
Installation Command:
code --add-mcp '{"name":"playwright","command":"npx","args":["@playwright/mcp"]}'
Verification
Ask Copilot: "What browser tools do you have access to?"
