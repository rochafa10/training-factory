Setting Up VS Code

Week 3 · GitHub Copilot & Development Tools
SOURCE: code.visualstudio.com/docs/getstarted
1

Download & Install

Get VS Code from code.visualstudio.com. Small download (<200 MB), runs on Windows, Mac, Linux.

2

Open a Folder

File → Open Folder to create a workspace. VS Code remembers your layout per folder.

3

Explore the Interface

Activity Bar (left) switches views. Editor (center) for code. Terminal (bottom) for commands.

4

Install Extensions

Search the marketplace for Python, GitHub Copilot, and other tools you need.

5

Command Palette

Press Ctrl+Shift+P to access any command. This is your power shortcut.
