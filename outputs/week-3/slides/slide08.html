Exercise 2

Voice Mode Setup

1. Install VS Code Speech

Extensions (Ctrl+Shift+X)
Search "VS Code Speech"
Install and reload

2. Configure Settings

Settings (Ctrl+,)
Search "accessibility.voice"
Enable wake word (optional)

3. Test Voice Input

Press Ctrl+U
Say "Hello, can you hear me?"
Press Escape to stop

Voice Commands

Ctrl+U Voice Chat
Ctrl+Alt+V Editor Dictation
Escape Stop Dictation

Voice is 3-5x faster than typing. Humans speak at 150 WPM vs 40 WPM typing.
