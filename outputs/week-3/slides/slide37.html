Common Diff Patterns

1
Simple Addition
- Headcount Summary + - Overtime Analysis + - Safety Incidents - Action Items

2
Replacement
# Weekly Report Template - Last updated: January 2026 + Last updated: February 2026 ## Sections

3
Deletion
## Sections - - Deprecated Feature - Current Features - Action Items

4
New File
+ # New Template + + This is a new file created by the agent. + Use it for shift handoffs.
What happened: All green — entire file is new
U = Untracked (new file) • M = Modified (existing file changed)
