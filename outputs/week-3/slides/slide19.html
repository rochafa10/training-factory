Terminal Command Approval — Safety First

✅
Safe to Approve
  • mkdir create folder
  • touch create file
  • git status, git log, git add
  • ls, pwd, dir
⚠
Review Carefully
  • rm, del deleting files
  • curl, wget downloading
  • npm install, pip install software
  • Anything you don't understand
🛑 Rule: If you wouldn't type it yourself, don't approve it.
Never click "Continue" without reading what the agent plans to do.
