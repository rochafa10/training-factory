The ReAct Pattern - How Agents Think

1
Think
Agent analyzes what needs to happen
"I need to create 3 files: template, sample, README"
2
Act
Agent takes one step
Creates the template file
3
Observe
Agent checks if it worked
"File created successfully. Moving to next step."
4
Repeat
Back to step 1 until goal is met
Analogy
Like a new hire who thinks before acting and checks their work after each step
Think → Act → Observe ↻
You describe the goal. The agent figures out the steps.
