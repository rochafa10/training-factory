Git Review Checklist

1
Open Source Control
Or click the branch icon in the sidebar
Ctrl+Shift+G
2
Check file list
Look for U (new) and M (modified) markers
3
Click each file
See full diff view with green/red highlighting
4
Verify content matches request
All sections present? No Tesla data?
5
Stage or Discard
Right-click > "Stage Changes" (accept) or "Discard Changes" (reject)
6
Ask Copilot if unclear
"Explain what [filename] does"
Never approve changes you don't understand.
Review agent work like you'd review an associate's work — before it goes out.
