# AI Tools for Operations Leaders
## 4-Week Training Program for Tesla AGMs/GMs

---

## Course Overview

**Instructor:** Fabricio Rocha, Sr. Regional Manager, Supply Chain Operations  
**Audience:** 5 Area General Managers (AGMs) across East Coast Distribution Centers  
**Duration:** 4 weeks (90 min/session + async homework)  
**Primary Tools:** Bottle Rocket (go.tesla.com/chat) & GitHub Copilot in VS Code

---

## Learning Objectives

By the end of this course, participants will be able to:
1. Use Tesla-approved AI tools in compliance with company policy
2. Write effective prompts that generate useful, actionable outputs
3. Apply AI to daily operations tasks (communications, analysis, planning)
4. Use GitHub Copilot Agent Mode for basic automation tasks
5. Evaluate AI outputs critically and iterate for better results

## Week 4: Agentic Automation with GitHub Copilot

### Learning Objectives
- Use Agent Mode for autonomous task completion
- Build simple automation scripts for operations tasks
- Understand the plan-execute-iterate loop
- Apply AI to real supply chain scenarios

### Session Agenda (90 min)
1. Agent Mode Deep Dive (20 min)
2. Live Demo: Building an Automation (25 min)
3. Hands-On Lab: Your First Agent Task (30 min)
4. Course Wrap-Up & Next Steps (15 min)

### Key Topics
- How Agent Mode works (plan → execute → iterate)
- Approving terminal commands safely
- Multi-file editing capabilities
- MCP tools (approved Nova tools only)
- Limitations and best practices

### Hands-On Labs

**Lab 1: CSV Data Analyzer**
Create a script that:
- Reads daily shipment data from CSV
- Calculates key metrics (avg, min, max, trends)
- Generates a summary report
- Flags anomalies

**Lab 2: Email Draft Generator**
Create a script that:
- Takes input parameters (recipient, topic, tone)
- Generates professional email draft
- Saves to a text file

**Lab 3: Meeting Notes Formatter**
Create a script that:
- Takes raw meeting notes as input
- Extracts action items
- Formats into structured summary
- Outputs to markdown file

### Final Assignment
Build one automation tool for your DC using Agent Mode:
- Define the problem and desired output
- Document the prompts used
- Test and iterate
- Present to the group in Week 4 follow-up session

## Policy Compliance Quick Reference

### ✅ APPROVED - Tesla Data Allowed
- Bottle Rocket (go.tesla.com/chat)
- IT Assist Bot
- Employee Assist Bot

### ⚠️ CONDITIONALLY APPROVED - No Tesla Data
- GitHub Copilot (general coding only)
- GitHub Copilot CLI

### ❌ PROHIBITED
- ChatGPT, Claude.ai, Gemini (external)
- Meeting transcription/recording AI
- Apple Intelligence on corporate devices
- Any tool not on go.tesla.com/aitools

### What is "Tesla Data"?
Any information about Tesla, customers, employees, affiliates, or users including:
- Performance metrics, KPIs
- Employee names, schedules, personal data
- Customer information
- Internal processes, SOPs
- Financial data
- Proprietary code
